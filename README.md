# ui-stencil-demo

> A UI demo

## Build Setup

``` bash
# 在react中使用ui库
cd ui-components-react
yarn link      // 创建软连接

cd react-pro
yarn install
yarn link @ai.s/ui-components-react
yarn start

# 在vue中使用ui库
cd ui-components-vue
yarn link    // 创建软连接

cd vue-pro
yarn install
yarn link @ai.s/ui-components-vue
yarn start

# 在angular中使用ui库
cd ui-components-angular
yarn link    // 创建软连接

cd ng-pro
yarn install
yarn link @ai.s/ui-components-angular
yarn start
```
